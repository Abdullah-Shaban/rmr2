# K-means

##


```r
    dist.fun = 
      function(C, P) {
        apply(C,
              1, 
              function(x) 
                matrix(
                  rowSums((t(t(P) - x))^2), 
                  ncol = length(x)))}
```

 
##

```r
    kmeans.map.1 = 
      function(., P) {
        nearest = 
          if(is.null(C)) {
            sample(
              1:num.clusters, 
              nrow(P), 
              replace = T)}
          else {
            D = dist.fun(C, P)
            nearest = max.col(-D)}
        keyval(nearest, P) }
```


##

```r
    kmeans.reduce.1 = 
      function(x, P) {
        t(as.matrix(apply(P, 2, mean)))}
```


##


# Linear Least Squares

##
  
  $$ \mathbf{X b = y}$$  

```
  solve(t(X)%*%X, t(X)%*%y)
```
  
##


```r
Sum = 
  function(., YY) 
    keyval(1, list(Reduce('+', YY)))
```


##

```r
XtX = 
  values(
    from.dfs(
      mapreduce(
        input = X,
        map = 
          function(., Xi) 
            keyval(1, list(t(Xi) %*% Xi)),
        reduce = Sum,
        combine = TRUE)))[[1]]
```
